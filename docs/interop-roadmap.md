# Interop roadmap

Phase 1:

- shared CLI schema parity
- normalized JSON parity
- shared fixtures
- subprocess IPC over stdin/stdout with JSON lines

Phase 2:

- targeted FFI where natural
- JVM interop for Java and Kotlin
- native ABI lanes where justified
- NATS-backed orchestration for distributed fixture generation
- optional GraalVM research in the interop lane only
